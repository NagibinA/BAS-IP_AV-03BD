# BAS-IP Integration for Home Assistant

Интеграция для управления IP-домофоном BAS-IP через Home Assistant.

## Возможности
- Открытие замка
- Просмотр с камеры
- Аварийное открытие/закрытие
- Перезагрузка устройства
- Управление языком интерфейса
- Настройка сети (статический IP / DHCP)
- И многое другое...

## Установка через HACS
1. Добавьте репозиторий `https://github.com/NagibinA/BAS-IP_AV-03BD` как пользовательский репозиторий в HACS.
2. Установите интеграцию "BAS-IP Intercom".
3. Перезапустите Home Assistant.
4. Добавьте интеграцию через `Настройки` -> `Устройства и сервисы` -> `Добавить интеграцию` -> `BAS-IP`.
5. Введите IP-адрес и пароль вашего устройства.

## Настройка
После установки введите IP-адрес и пароль от панели BAS-IP.

## Поддерживаемые устройства
- BAS-IP AV-03BD
- BAS-IP AV-04BD (и другие модели на этом API)
